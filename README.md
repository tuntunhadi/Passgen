# PassGen

Offline password generator PWA built with Next.js.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project
3. Import your GitHub repo
4. Klik Deploy — done ✅

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Install as PWA

Setelah deploy, buka URL Vercel di browser/HP → klik "Add to Home Screen".
